Local Events App

Description

Mobile friendly web app for discovering local events. Ability to view events in Google Maps and add them to Google Calendar.

Integrated with 3rd-party API to provide event data. Utilized browser geolocation services combined with Google Maps API to create custom user experience.

Technology

  • Angular
  • Node.js
  • Express
  • Google Maps
  • Google Calendar
  • Typescript
  • Javascript
  • HTML / CSS
